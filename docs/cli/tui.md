# `verso tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
verso tui
verso tui --url ws://127.0.0.1:18789 --token <token>
verso tui --session main --deliver
```
