# `verso setup`

Initialize `~/.verso/verso.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
verso setup
verso setup --workspace ~/verso
```

To run the wizard via setup:

```bash
verso setup --wizard
```
